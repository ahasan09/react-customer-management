# React App

React 16 Create React App application for customer management with client-side routing and Bootstrap styling.

## Tech Stack
- React 16
- React Router
- Bootstrap
- JavaScript (ES6+)

## Project Structure
```
react-app/
├── src/
│   ├── components/
│   └── App.js
├── public/
└── package.json
```

## Development
```bash
# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
```
